Tipo Number:

let numero = 5; // Vamos multiplicar o valor por 5

numero * 5

Tipo String:

let nome = 'WiLL' sobrenome = 'Sposito'; // Vamos Concatenar as variaveis WiLL + Sposito

nome + sobrenome

Tipo Booleano:

nome == 'WiLL'; // Será verificado se a variavel nome ta escrito WiLL, caso seja verdadeiro o valor retorna true, caso for falso retorna false

nome == 'WiLL'

Atributo id:

Esse atributo insere no elemento um identificador único

Atributo Class:

O atributo class assim como o id irá facilitar nossas vidas

Atributo href:

Você deve se lembrar desse atributo quando trabalhamos com o elemento

Atributo src:

Esse atributo é um pouco parecido com o href,

Atributo title:

Esse atributo é bem bacana e é usado para facilitar a vida do usuário

Atributo alt:

Como vimos na tag , o atributo alt tem o papel de mostrar um texto alternativo a imagem

Atributo width:

Falando em imagens aqui temos um atributo que pode ser bastante útil para redimensionar uma imagem rapidamente
